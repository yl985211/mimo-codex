# MiMo-Codex v8.0.0

MiMo-Codex v8 是基于 Claude Code Haha 的重大版本升级，深度适配小米 MiMo 大模型，全面优化用户体验。

## 主要更新

### MiMo 模型深度适配

- 内置 MiMo 模型提供商预设，支持 OpenAI 和 Anthropic 两种 API 格式
- 支持 `mimo-v2.5-pro` 和 `mimo-v2.5` 两款模型
- 支持 MiMo 官方 API 和 MiMo Plan 两种套餐

### 品牌重塑

- 全新应用名称：MiMo-Codex
- 全新应用图标和 UI 色系（小米橙金风格）
- 独立的应用数据存储（`.mimo-codex` 文件夹）

### 架构优化

- Sidecar 动态加载外部依赖，减小打包体积
- 支持与 Claude Code Haha 同时运行
- 配置数据便携化，存储在可执行文件同目录

### 服务商预设

| 提供商 | API 格式 | Base URL |
|--------|----------|----------|
| MiMo (OpenAI 兼容) | OpenAI Chat | `https://api.xiaomimimo.com/v1/chat/completions` |
| MiMo (Anthropic 兼容) | Anthropic | `https://api.xiaomimimo.com/anthropic/v1/messages` |
| MiMo Plan (OpenAI 兼容) | OpenAI Chat | `https://token-plan-cn.xiaomimimo.com/v1` |
| MiMo Plan (Anthropic 兼容) | Anthropic | `https://token-plan-cn.xiaomimimo.com/anthropic` |

## 安装

到本页面底部的 Assets 里，按系统和 CPU 架构下载对应文件：

| 平台 | 文件 |
|------|------|
| macOS Apple Silicon（M 系列） | `MiMo-Codex-8.0.0-mac-arm64.dmg` |
| macOS Intel | `MiMo-Codex-8.0.0-mac-x64.dmg` |
| Windows x64 | `MiMo-Codex-8.0.0-win-x64.exe` |
| Linux x64 | `MiMo-Codex-8.0.0-linux-x86_64.AppImage` 或 `MiMo-Codex-8.0.0-linux-amd64.deb` |
| Linux ARM64 | `MiMo-Codex-8.0.0-linux-arm64.AppImage` 或 `MiMo-Codex-8.0.0-linux-arm64.deb` |

### macOS

双击 DMG 把应用拖入「应用程序」。首次打开如果提示"已损坏"或"无法验证开发者"，在终端执行：

```bash
xattr -cr /Applications/MiMo-Codex.app
```

也可以在「系统设置 → 隐私与安全性」里点"仍要打开"。

### Windows

双击 `.exe` 安装。SmartScreen 提示时点「更多信息」→「仍要运行」。

### Linux

AppImage：

```bash
chmod +x MiMo-Codex-8.0.0-linux-x86_64.AppImage
./MiMo-Codex-8.0.0-linux-x86_64.AppImage
```

deb：

```bash
sudo apt install ./MiMo-Codex-8.0.0-linux-amd64.deb
```

ARM64 机器把文件名换成 `...-linux-arm64.AppImage` / `...-linux-arm64.deb`。

## 从源码构建

```bash
# 克隆仓库
git clone https://github.com/yl985211/mimo-codex.git
cd mimo-codex

# 安装依赖
bun install
cd desktop && bun install && cd ..

# 构建桌面应用
cd desktop
bun run build
bun run build:electron

# 构建 Sidecar
bun build ./sidecars/claude-sidecar.ts \
  --compile \
  --target bun-windows-x64-baseline \
  --outfile ./src-tauri/binaries/claude-sidecar-x86_64-pc-windows-msvc.exe \
  --minify

# 打包
bunx electron-builder --dir --publish never
```
